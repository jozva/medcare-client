MedCare is a full-stack healthcare management web application where doctors can manage patients, track vitals, medications, and upcoming tests, while patients can view their health data and track daily medication intake.

This project is built using the MERN-style stack with Supabase as the database.

---
Features

### Doctor
- Login as doctor
- View all patients
- Add new patients
- Edit patient vitals
- Add, update, delete medications
- Add, update, delete upcoming tests
- View patient medication calendar

### Patient
- Login as patient
- View personal dashboard
- View vitals
- View medications
- View upcoming tests
- Mark medication as taken
- Calendar showing medication adherence

---

Tech Stack

### Frontend
- React.js
- React Router
- Tailwind CSS
- React Calendar

### Backend
- Node.js
- Express.js

### Database
- Supabase (PostgreSQL)
